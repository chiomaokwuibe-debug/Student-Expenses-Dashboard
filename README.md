# Student-Expenses-Dashboard
Description: "Power BI dashboard showcasing analysis of student expenses, spending behavior, and budget trends through interactive visualizations."
Student Expenses Dashboard
Tools Used: Power BI
Project Overview:
This dashboard analyzes student spending patterns across different expense categories. It provides insights into spending behavior, budget allocation, and financial trends using interactive visualizations.
Key Features:
Expense category analysis
Spending trend visualization
Interactive filters and charts
Financial insights for decision-making
